## Pikachu Jump

    Joguinho com tématica de pókemon. Baseado no "jogo do dinossauro offline" do chrome

## Funcionalidades

    - Score Tracking
    - Score Management
    - Sistema de dia/noite apartir do score

## Tecnologias Usadas

    - HTML
    - CSS
    - JavaScript
